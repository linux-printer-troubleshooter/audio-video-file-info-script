# audio-video-file-info-script
Returns Useful Stats file information about your file from ffmpeg. (resolution, number of streams, bitrate )
## Sample output
```
return_info.sh ./My\ \(Mooney\).mkv

Resolution:1280x720
audio:4
video:1
Bitrate:48000-Hz
Bitrate:48000-Hz
Bitrate:48000-Hz
Bitrate:48000-Hz
Stream Map:
0:0,Video
0:1,Audio
0:2,Audio
0:3,Audio
0:4,Audio
0:5,Subtitle


```
